# Smart Employee & Expense Management System

A backend Employee & Expense Management System built using **Python** and **Django REST Framework**.

## Features

- Employee Management
- Department Management
- Expense Management
- JWT Authentication
- REST APIs
- Expense Approval Workflow
- Search & Filtering
- Pagination
- Dashboard APIs
- Git Version Control

## Technologies

- Python
- Django
- Django REST Framework
- SQLite (PostgreSQL Coming Soon)
- JWT Authentication
- Git
- GitHub

## API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /api/employees/ | List Employees |
| POST | /api/employees/ | Create Employee |
| GET | /api/departments/ | List Departments |
| POST | /api/departments/ | Create Department |
| GET | /api/expenses/ | List Expenses |
| POST | /api/expenses/ | Create Expense |
| POST | /api/token/ | Generate JWT Token |
